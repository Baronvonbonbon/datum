Polkadot Hub Launches Privacy-Preserving Ad Network

DATUM, a decentralized advertising protocol built on Polkadot Hub, has launched its alpha testing phase. The protocol uses on-chain settlement with privacy-preserving impression tracking, giving users control over their data while compensating publishers and users for ad engagement.

Publishers integrate using a lightweight SDK tag on their site. The SDK establishes a challenge-response handshake with the DATUM browser extension to verify legitimate impressions. Campaigns can target specific publishers or run as open campaigns matched via category overlap.

How to Integrate the DATUM SDK

Add the following snippet to your page:

<script src="datum-sdk.js"
  data-categories="6,26"
  data-publisher="0xYOUR_ADDRESS"></script>
<div id="datum-ad-slot"></div>

The data-categories attribute declares which ad categories your site accepts (comma-separated IDs from the DATUM taxonomy). The data-publisher attribute is your registered on-chain publisher address.
