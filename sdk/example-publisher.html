Polkadot Hub Launches Privacy-Preserving Ad Network

DATUM, a decentralized advertising protocol built on Polkadot Hub, has launched its alpha testing phase. The protocol uses on-chain settlement with privacy-preserving impression tracking, giving users control over their data while compensating publishers and users for ad engagement.

Publishers integrate using a lightweight SDK tag on their site. The SDK establishes a challenge-response handshake with the DATUM browser extension to verify legitimate impressions. Campaigns can target specific publishers or run as open campaigns matched via tag overlap.

How to Integrate the DATUM SDK

Add the following snippet to your page:

<script src="datum-sdk.js"
  data-tags="crypto-web3,en,desktop"
  data-publisher="0xcA5668fB864Acab0aC7f4CFa73949174720b58D0"
  data-relay="https://your-relay.example.com"></script>
<div id="datum-ad-slot"></div>

The data-tags attribute declares which tags describe your site (comma-separated values from the DATUM tag dictionary). The data-publisher attribute is your registered on-chain publisher address. The data-relay attribute points to your publisher relay endpoint URL (the extension picks this up automatically — no rebuild needed when the URL changes).
