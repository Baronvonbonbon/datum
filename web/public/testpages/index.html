How to run the manual e2e test

  1. Make sure the local IPFS node is serving creatives (http://127.0.0.1:8090/ipfs/) and, to settle, the lab relay is running (http://127.0.0.1:3400).
  2. Build + load the DATUM extension (alpha-core/extension) pointed at the alpha-5 v5 addresses.
  3. Open any site above. The SDK declares its ad slots; the extension classifies the page, runs a second-price auction over the active campaigns, fetches the winning creative from IPFS, and injects it at the exact IAB size.
  4. Claims auto-submit to the relay, which co-signs and settles on-chain. Watch the indexer (datum-labs/indexer) for ClaimSettled.
